# Student Performance Analyzer (JavaScript)

This is a console-based JavaScript project created to practice and demonstrate
core JavaScript fundamentals.

## Concepts Covered
- Variables and Data Types
- Operators and Conditional Statements
- Loops and Strings
- Arrays and Objects
- Functions and Methods

## Features
- Stores student names and marks
- Calculates grades based on marks
- Calculates average marks
- Finds highest and lowest marks
- Displays results in the console

## How to Run
1. Open the project folder
2. Run `script.js` in browser console or VS Code terminal

## Note
This project does not use DOM or UI.
It focuses purely on JavaScript logic.
